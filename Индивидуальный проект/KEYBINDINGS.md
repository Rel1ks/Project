# Горячие клавиши редактора

## Навигация

- `Up` / `Down`: перейти на визуальную строку выше/ниже (с учетом soft wrap).
- `Left` / `Right`: перемещение по символам.
- `Home`: в начало текущей визуальной строки.
- `End`: в конец текущей визуальной строки.
- `PageUp` / `PageDown`: перейти на экран вверх/вниз.

## Ввод и редактирование

- `Enter`: новая строка.
- `Backspace`: удалить символ слева от курсора.
- `Delete`: удалить символ справа от курсора.
- `Tab`: вставить 4 пробела.
- Любая печатная клавиша: вставка текста.
- `Ctrl+A`: выделить весь текст.
- `Ctrl+Z`: Undo.
- `Ctrl+Y`: Redo.

## Файлы

- `Ctrl+S`: сохранить текущий файл (или запросить имя, если файл еще не задан).
- `Ctrl+Shift+S` (Windows): `Save As`.
- `Ctrl+T` (Linux/blessed): `Save As`.
- `F2`: `Save As`.
- `Ctrl+O`: открыть файл.
- `Ctrl+Q`: выход (с подтверждением при несохраненных изменениях).
- `Esc` (Windows): выход (с подтверждением при несохраненных изменениях).

## Поиск

- `Ctrl+F`: открыть поиск; при повторном нажатии очищает текущий поиск.

## Дополнительные последовательности через `Esc` (Linux ветка)

- `Esc`, затем `n`: следующий результат поиска.
- `Esc`, затем `/`: открыть prompt поиска.
- `Esc`, затем `s`: `Save As`.
- `Esc`, затем `o`: открыть файл.
- `Esc`, затем `q`: выход.

## Windows-специфично

- `Ctrl+Left` / `Ctrl+Right`: переход по словам.
- `Ctrl+Up`: перейти в начало файла.
- `Ctrl+Down`: перейти в конец файла.

## В режиме prompt (Open/Save/Search)

- `Enter`: подтвердить ввод.
- `Esc`: отменить.
- `Left` / `Right`: перемещение курсора внутри строки ввода.
- `Backspace`: удалить символ слева.
- `Tab`: автодополнение пути.
